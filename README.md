# 济幼社团 — 智慧社团活动综合管理系统

> 面向高校社团的全流程智慧管理平台，深度融合 AI 智能助手、自动化 Agent 与多角色协同，实现从活动策划到学分认定的全链路数字化闭环。

---

## 项目亮点

### 创新性（30%）

- **AI 智能助手"小通"**：全站浮动聊天入口，支持上下文感知（根据当前页面智能推荐操作），具备 6 种工具调用能力（审批查询、文件搜索、预警检查、数据报告、社团数据查询、社团列表），可获取真实业务数据并给出精准建议
- **AI Agent 自动化系统**：文档索引 Agent（自动索引上传文档，支持智能搜索）、通知 Agent（智能检查与发送）、数据洞察 Agent（自动生成分析报告）、审批 Agent（预览/分析/自动审批），构建无人值守的智能运营体系
- **AI 海报生成**：智能生成社团招募宣传海报，一键出图
- **智能社团推荐**：基于 17 种技能关键词库 + 多维度匹配（名称/描述/分类/部门/活动），计算匹配百分比并给出推荐理由和推荐部门

### 实用性（25%）

- **四角色权限体系**：管理员、社团负责人、指导老师、学生 — 各有专属导航与功能，互不干扰又紧密协同
- **全流程活动管理闭环**：活动策划 → 签到考勤（签到码 + 扫码 + GPS定位） → 材料上传 → 审批流转 → 赋分计算 → 学分认定，环环相扣
- **三级赋分审核流程**：社团提交 → 指导老师审核 → 团委审核，确保赋分公平透明
- **财务精细化管理**：收支记录、分类统计、附件上传、财务管理员授权机制
- **协助工具箱**：投票（单选/多选/匿名/限员）、问卷、报名表 — 一键创建，结果导出 Excel
- **校外活动审批模板**：上传 docx 模板 → 自动填充 → 提交审批，规范校外活动管理
- **树洞匿名社区**：公开/社团内部两种范围，保护学生隐私的同时促进交流
- **联合活动平台**：跨社团发布合作邀请、留言互动、合作确认

### 技术实现（20%）

- **后端**：Python Flask，约 15800 行代码，200+ API 端点
- **数据库**：SQLite，38+ 张业务表，覆盖用户、社团、活动、签到、赋分、财务、通知、AI 等全领域
- **前端**：纯原生 HTML/CSS/JS 单页应用，零框架依赖，Chart.js 图表，Quill.js 富文本，jsQR 扫码
- **数据引擎**：自实现 `DataCleaner`（去重/空值/规范化）+ `StatsService`（描述性统计）+ LRU 缓存（带 TTL）
- **文件存储**：本地存储与阿里云 OSS 双模式无缝切换，自动迁移旧路径
- **AI 集成**：多 LLM Provider 支持（通义千问等），工具调用协议，意图检测回退机制
- **文档解析**：支持 Excel（Pandas/openpyxl）、Word（python-docx）、PDF（PyMuPDF）多格式处理
- **响应式设计**：完整的移动端适配（768px/480px 断点），触控优化

### 用户体验（15%）

- **角色主题差异化**：管理员粉色、学生绿色、老师金色、负责人紫色 — 登录即感知身份
- **移动端优先**：全部页面均有完善的手机端适配，表格横向滑动、按钮自适应、输入框触控友好
- **一键操作**：签到码输入 + 摄像头扫码双模式，教师 GPS 签到/签退
- **实时反馈**：Toast 提示、加载动画、状态标签（待审核/已通过/已驳回）
- **智能引导**：AI 助手根据当前页面上下文推荐操作，降低学习成本
- **数据可视化**：Chart.js 图表展示活动统计、成员分布、财务趋势

### 商业价值（10%）

- **可复用的 SaaS 架构**：多社团/多角色/多学院的设计天然支持多租户扩展
- **AI 能力商品化**：AI 助手、智能审批等模块可独立拆分为增值服务
- **数据驱动决策**：低活跃预警、数据洞察报告 — 为管理者提供决策依据
- **降低管理成本**：自动化审批、智能通知、批量操作 — 显著减少人工管理负担
- **提升学生参与度**：树洞社区 + 智能推荐 — 激活校园社团生态

## 快速开始

### 环境要求

- Python 3.8+
- pip

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/wolq11/-.git
cd club-stats

# 安装依赖
pip install -r requirements.txt

# 启动服务
python server.py
```

服务默认运行在 `http://localhost:5000`

### 首次使用

1. 访问登录页，选择角色注册（管理员/社团负责人/指导老师/学生）
2. 管理员可创建社团、设置赋分规则、管理用户
3. 社团负责人可完善社团信息、开展活动、上传材料
4. 指导老师可审核赋分、签到签退
5. 学生可浏览社团、报名参加、签到打卡

## 数据库设计

系统包含 38+ 张业务表，核心实体关系：

- **用户体系**：users → user_profiles / teacher_profiles → club_members / club_cadres
- **社团体系**：club_profiles → club_departments → club_teachers → club_members
- **活动体系**：checkin_sessions → checkin_records / teacher_checkin_checkout → activity_records
- **赋分体系**：scoring_rules → scoring_submissions → scoring_submission_items → final_credits
- **财务体系**：finance_records → finance_managers → finance_permissions
- **AI 体系**：ai_chat_history / doc_index / doc_embeddings
